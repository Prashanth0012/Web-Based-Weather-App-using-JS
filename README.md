# Weather App

## Overview
This project is a simple Weather Application built using **HTML, CSS, and JavaScript**.  
The application allows users to enter a city name and fetch real-time weather information using the **OpenWeatherMap API**.

The app displays important weather details such as temperature, weather condition, and humidity in a clean user interface.

---

## Features

- Search weather by city name
- Fetch real-time weather data using API
- Display temperature in Celsius
- Show weather condition and humidity
- User-friendly interface
- Error handling for invalid city names

---

## Technologies Used

Frontend
- HTML
- CSS
- JavaScript

API
- OpenWeatherMap API

---

## How It Works

1. User enters a city name in the input field.
2. The application sends a request to the OpenWeatherMap API.
3. The API returns weather data for the city.
4. JavaScript processes the response and displays:
   - City name
   - Temperature
   - Weather description
   - Humidity

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run the `weather.html` file in your browser
4. Enter a city name and click **Get Weather**
